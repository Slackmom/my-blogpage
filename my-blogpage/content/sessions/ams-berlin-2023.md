## What This Session Was About

Microsoft Mesh opens up a new design space — immersive 3D environments for meetings, training, and collaboration. But until visual scripting arrived, building these experiences required developer expertise most teams don't have.

This session showed attendees how visual scripting in Mesh puts creation in the hands of designers, trainers, and change agents — not just developers. Through live demos, we built interactive 3D spaces and showed what's possible when the barrier to entry drops.

## Key Topics

- How visual scripting works in Microsoft Mesh
- Building interactive elements without code
- Real use cases: onboarding, training, and team events in virtual spaces
- What makes Mesh different from previous metaverse attempts
